# JobTracker — Client

React (Vite + Tailwind) frontend for JobTracker. See the [root README](../README.md) for
what this project is and how to run the whole thing.

## Dev commands

```bash
npm install
npm run dev       # local dev server with hot reload
npm run build     # production build, output in dist/
npm run lint      # oxlint
```

Requires a `.env` (see `.env.example`) pointing `VITE_API_URL` at the running server.
